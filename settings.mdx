## **API Keys** 

Generate and manage API keys for programmatic access to Q0 models and services

<Frame>
  ![Image](/images/image-89.png)
</Frame>

- Displayed at the top of the API Keys tab, the Current Playground Token is the active token used by the Q0 Playground. 

- This token is auto-rotated by Q0 — you do not need to manage it manually.

**Generate API Keys** 

- Click on the Generate API Keys button 

- Set Key Name 

- Select Expiration from drop down, can be set to a week, Month, 3 Months and 6 Month 

## **Quota** 

View your TPM (Tokens Per Minute) and RPM (Requests Per Minute) limits and current usage across all Q0 models and services

<Frame>
  ![Image](/images/image-90.png)
</Frame>

| **Column** | **Description** |
| :-- | :-- |
| Resource/Service | The model or service name with its avatar icon and a short ID below it (copyable) |
| TPM — Current Limit & Usage | Tokens Per Minute. Shows 'Spent N' (current usage) and 'Current Limit NK' (your allowance). Progress bar below. |
| RPM — Current Limit & Usage | Requests Per Minute. Same layout as TPM — Spent and Current Limit with progress bar. |

For Individual Resource/Services, set **Increase TPM Limit** or **Increase RPM Limit**

## **Reservation** 

Reserve dedicated GPU capacity on Yotta Shakti Cloud for your Q0 workloads — guaranteed compute, no queuing

<Frame>
  ![Image](/images/image-21.png)
</Frame>

### **Create New Reservation**

| **Field** | **Confirmed options / behaviour** | **Notes** |
| :-- | :-- | :-- |
| Accelerator Type | Dropdown — confirmed: H100 (NVIDIA) ₹432/GPU/Hr (8 GPU's, 1.40 GHz, 80.00GB HBM3) and L40S (NVIDIA) ₹190.79/GPU/Hr (4 GPU's, 1.90 GHz, 48.00GB GDDR6 ECC) | Select the GPU type to reserve |
| Accelerator Count | Button group: Auto, 1, 4, 8, 16, 24, 32, Custom | Number of GPUs to reserve. Custom allows entering a specific count. |
| Region | Dropdown — 'Select region' (options confirmed: NM1 / DL1 from other screens) | The Yotta Shakti Cloud region for the reservation |
| Start Date | Date picker (calendar icon) | The date the reservation begins |
| End Date | Date picker (calendar icon) | The date the reservation ends |
| Total | Read-only field — shows '-- /week' until all fields are filled | Calculated weekly cost based on GPU type, count, and duration |
