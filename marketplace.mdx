Discover 100\+ AI models across text generation, speech, image, and more — filter by task or publisher, try them instantly, and deploy to dedicated GPUs 

The **Marketplace** is the central hub on Q0 for discovering and working with AI models. It brings together 100\+ models from leading providers (Meta, OpenAI, DeepSeek, and more) into a single catalog, organized by category, so you can quickly find the right model for your use case — whether that's text generation, text-to-speech, image generation, or other modalities.

**From the Marketplace, you can: ** 

- Browse models by category (Text Generation, Text to Speech, Image Models, and more)  

- Filter and sort the catalog to narrow down your search  

- View detailed specs, pricing, and supported features for any model  

- Try a model directly in the Playground, deploy it on demand, or fine-tune it on your own data

A **Global search bar** ("Discover 100s models and try out what interests you!") that lets users search the entire catalogue by model name or keyword, independent of category filters. 

A **Filters** button opens an advanced filter panel. This is intended for multi-attribute filtering — for example, narrowing by provider, pricing range, context length, or supported features — in addition to category. 

- Pill-shaped tabs lets users quickly switch between top-level model categories 

- **Sort by** dropdown, anchored to the top-right of the catalog section, allows users to reorder the visible mode l cards (e.g. Popularity, or New Model). 

Each card in the grid represents a single model and displays the following fields: 

<Frame>
  ![Image](/images/image-8.png)
</Frame>

- Provider logo - Identifying the model's source/developer (e.g., Meta, OpenAI, DeepSeek) 

- “New" badge - Optional green badge highlighting recently added models 

- Model Name - Model's display name (e.g. Llama3-1-8B, GPT-OSS-20B, DeepSeek-R1-Distill-Llama-70B) 

- Category tag -  A pill restating the model's category (e.g., \`Text Generation 

- Input price - Cost per 1 million input tokens, shown in INR (e.g., `₹190`). 

- Output price -  Cost per 1 million output tokens, shown in INR (e.g., `₹220`).  

- Context window- Maximum context length the model supports, shown in tokens (e.g., `128K`, `131K`, `8K`).  

Clicking anywhere on a model card navigates the user to that model’s **Model Detail Page**

For any model page shows  

- **Short Description about Model** - A short paragraph describing what the model is, its lineage, and what it's optimized for (e.g., instruction tuning, multilingual dialogue, benchmark performance relative to other models in its family). 

- **Model Features -** A set of feature cards (typically three) highlighting the model's standout capabilities. Each card includes an icon, a title, a short description, and an **Explore →** link to learn more. Examples seen for Llama3-1-8B: 

- **API Documentation -** An interactive code panel showing how to call the model programmatically. Send requests with model, prompt, and parameters in JSON, and the system calculates usage costs based on the input and output units processed
